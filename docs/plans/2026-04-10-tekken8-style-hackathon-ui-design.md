# Tekken8 风格黑客松 UI 设计

## 概述

将黑客松模拟的左侧 RosterGrid 组件改造为铁拳8官网风格的平行四边形卡牌系统，右侧 ChatSidebar 做配色统一优化。

## 视觉风格

- **配色**：深黑背景 `#020108`，青色高光 `#3fd1e7`，粉红标签 `#f50a64`
- **形状**：`skewX(-28deg)` 平行四边形，图片反向 `skewX(28deg)` 矫正
- **交互**：hover 时青色边框 + 光晕（`box-shadow: 0 0 15px rgba(63,209,231,0.6)`），暗色遮罩移除

## Phase 0 — 初始展示

- 40 个角色密排平行四边形网格（左侧面板 65%）
- 不区分软件/硬件赛道，统一展示
- 逐个出场激活动画（80ms 间隔 stagger）
- 非 hover 卡牌暗色遮罩（`::before` opacity 0.6）
- hover：青色边框光晕 + 底部粉红名字标签滑出

## Phase 0 — 分组动画

- FLIP 技术：记录位置 → 重排为分组布局 → 1.2s 过渡动画
- 角色从随机网格位置平滑移动到对应组

## Phase 1+ — 分组后滚动布局

- 每组变为一个横向容器
- 内部：成员平行四边形大头照从右向左连续滚动（CSS marquee 动画）
- 元素复制一份实现无缝循环
- hover 暂停滚动（`animation-play-state: paused`）+ 显示角色信息弹出
- 活跃组：青色边框高亮
- 发言角色：特殊光效

## ChatSidebar 配色优化

- 组 Tab：选中态青色边框，未选中暗灰
- 队长标识：粉红徽章
- 发言头像：青色光晕边框
- 消息背景：深色系 `#0a0a14`
- 角色名：青色
- Role 标签：暗底 + 青色文字
- 未读徽标：粉红色
- 整体背景：统一 `#020108`

## 技术方案

- **方案 A（选定）**：纯 CSS 实现
- 平行四边形：CSS `skewX(-28deg)` + 反向矫正
- 滚动动画：`@keyframes marquee` + `translateX(-50%)`
- hover 暂停：`animation-play-state: paused`
- 分组 FLIP：复用现有逻辑
- 素材：现有 oc-1~40.webp + 参考前端部分素材

## 不变项

- 右侧 ChatSidebar 逻辑不变
- 后端逻辑不变
- 状态管理不变
- Phase 1-3 对话/创造/答辩流程不变
